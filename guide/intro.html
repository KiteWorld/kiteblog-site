# 什么是 KiteBlog ？

KiteBlog是本人工作之余写的，类似于掘金的社区项目，算是一个中型项目吧。主要分为三部分，也可以看成三个独立的项目：Node服务端、后台管理、前台页面（未开发）。

  • Node服务端：Node + Express + MySQL 「Github地址」 (opens new window) 「Gitee地址」 (opens new window)

  • 后台管理：Vue + ElmenetUI + Axios + js-cookies 「Github地址」 (opens new window) 「Gitee地址」 (opens new window)

  • 前台页面（未开发）：预计使用 NuxtJS 来开发 「Github地址」 (opens new window) 「Gitee地址」 (opens new window)

# 开发者

整个项目由本人独立开发完成，本人毕业、入职到现在差不多大半年了，工作经验不算长，主要是一个边学习边做的过程。难免有写得不好的地方和需要优化的点，也请多多包涵指正，谢谢。

如果你能从项目中学到点什么，不要忘了点个 "star⭐️"~!。鼓励一下我这个前端萌新，哈哈

# 项目预览

目前 Node服务端和后台管理已经部署到本人购买的 VPS ，方便大家预览！

由于服务器配置比较低，速度自然也比较慢。大家在预览的时候不要恶意添加大量数据占用服务器资源以及删除别人的数据。让我们做个文明的中国人，这里附上 极品MV (opens new window)，供大家欣赏，哈哈!
