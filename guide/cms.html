# 启动

# 克隆代码
git clone https://github.com/KiteWorld/KiteBlog_Front_Admin

# 进入项目根目录
cd KiteBlog_Front_Admin

# 安装依赖
npm i

#启动项目
npm run serve

# 项目特点

动态添加路由导航菜单（支持无限层级）、顶部 Tab（页面切换）、支持无限级分类、MarkDown编辑器、Echarts数据可视化（开发中）、 el-table表格组件二次封装....
