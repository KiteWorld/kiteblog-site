KiteBlog

一个类似于「掘金」的社区项目

项目预览 → (opens new window)

Node服务端

「KiteBlog」服务端，使用了 Node + Express + MySQL 的解决方案。项目特点： Token 生成验证、手写 SQL、文件上传、跨域配置、https 配置、websocket (配合数据可视化界面)、生成/开发环境配置...

管理后台

使用了 Vue + Vuex + ElementUI + Echarts + Axios + js-cookie 的解决方案。项目特点：动态添加路由导航菜单（支持无限层级）、顶部页签切换、支持无限级分类、MarkDown 编辑器、Echarts 数据可视化、 el-table 表格组件二次封装....

前台界面（未开发）

前台界面考虑到 SEO,决定使用 NuxtJS（学习中）来实现 SSR（服务端渲染）
