# 供应链管理系统（SCM）

供应链管理系统是一个基于 Vue 的大型项目，有十几个大的模块，每个大的模块下面又有十几个小模块。每个大模块功能数量以及复杂程度又相当于一个中大型的项目。

本人主要开发了三个模块：产品研发管理、供应商考核、精准来货，相当一个中大型项目、一个中型项目、中小型项目。本人主要负责前端部分的开发

  • # 产品研发管理

产品研发管理其实就是一个 PLM + WBS 的项目。主要功能就是跟踪产品研发的进度以及任务多级分解。

收获：这是本人入职以来第一个开发的项目，受益匪浅，第一次看到大型企业的项目全貌。无论项目全局观，还是技术细节，还是业务需求的分析，甚至是多人合作的开发模式等等，受益良多。但也深深得感受到身为外包人员无奈。例如：不敢对公用组件进行改造、不敢给项目添加新的插件或者依赖等等。这主要是为划清责任边界问题，所以只能重复"造轮子"，蹑手蹑脚，很多时候施展不开。

  • # 精准来货、供应商考核

这两个项目难度相对较小，加上经过开发「产品研发管理」已经比较熟悉供应链管理系统，很快就开发完成了。
